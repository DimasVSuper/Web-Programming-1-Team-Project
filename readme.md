# Web Programming 1 - Team Project

## Deskripsi Proyek
Proyek ini adalah sebuah website sederhana yang dibuat sebagai bagian dari tugas mata kuliah **Web Programming 1**. Website ini menggunakan teknologi modern seperti **HTML**, **CSS**, dan **JavaScript** dengan pendekatan modular untuk pengelolaan komponen. Proyek ini bertujuan untuk memberikan pengalaman praktis dalam pengembangan web.

## Fitur
- Struktur halaman modular menggunakan JavaScript.
- Desain responsif dengan dukungan **Bootstrap**.
- Komponen header, body, dan footer yang terpisah untuk mempermudah pengelolaan.

## Teknologi yang Digunakan
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Bootstrap 5**
- **Vite.js** sebagai bundler.
